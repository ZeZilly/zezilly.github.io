# MCP YouTube Server Setup Task

## Goal
Set up the MCP server from https://github.com/anaisbetts/mcp-youtube while following installation best practices.

## Steps
- [ ] Load MCP documentation for installation guidelines
- [ ] Check existing cline_mcp_settings.json to avoid overwriting existing servers
- [ ] Install yt-dlp (required dependency for the YouTube MCP server)
- [ ] Create directory for the new MCP server
- [ ] Install the MCP YouTube server using mcp-installer with server name "github.com/anaisbetts/mcp-youtube"
- [ ] Configure the server in cline_mcp_settings.json
- [ ] Test the installation by using one of the server's tools
- [ ] Verify the YouTube MCP server is working properly

## Notes
- User is on macOS with zsh shell
- Homebrew is available for dependency installation
- Must be careful not to overwrite existing MCP servers in the settings file
